# My Bitcoi Wallet

## Description
An blockchain wallet built using [Crypto.js](https://cryptojs.gitbook.io/docs/).

## Specifications
Wallet will be able to: 
- [ ] Generate keys
- [ ] Sign messages
- [ ] Verify signatures
- [ ] Encrypt messages
- [ ] Decrypt messages

### Stretch Goals
- [ ] Compatible with Bitcoin
- [ ] Compatible with Ethereum
- [ ] HD wallet
